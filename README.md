# python-security-script

## Objetivo

Desarrollar un algoritmo en Python que automatice la actualización de una lista de direcciones IP autorizadas, eliminando aquellas incluidas en una lista de revocación, con el fin de mejorar la gestión de control de acceso y reducir errores manuales.

### Habilidades Desarrolladas
- Manipulación de archivos en Python
- Uso de estructuras with para gestión segura de recursos
- Lectura y escritura de archivos (open, .read(), .write())
- Manipulación de cadenas y listas (.split(), .join())
- Uso de bucles for
- Implementación de estructuras condicionales
- Automatización básica de procesos de seguridad

### Herramientas utilizadas
- Python
